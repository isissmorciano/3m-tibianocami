# Esercizio 31: Elenco delle chiavi con .keys()

Crea un dizionario di libri e autori (es. {"1984": "Orwell", "Il Signore degli Anelli": "Tolkien", "Harry Potter": "Rowling"}). Usa il metodo `.keys()` (che restituisce una vista di tutte le chiavi del dizionario) per stampare un elenco di tutti i titoli dei libri, uno per riga.
