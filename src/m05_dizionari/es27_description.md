# Esercizio 27: Utilizzo di dizionari

Scrivi un programma Python che:

1. Crea un dizionario vuoto.
2. Aggiunge le seguenti coppie chiave-valore:
   - Chiave: "nome", Valore: il tuo nome
   - Chiave: "età", Valore: la tua età (come numero intero)
   - Chiave: "città", Valore: la tua città
3. Stampa i valori associati a ciascuna chiave.

Esempio di output:
```
Nome: Mario
Età: 30
Città: Milano
```
