# Esercizio 32: Coppie chiave-valore con .items()

Implementa un dizionario di città e popolazioni (es. {"Roma": 2870000, "Milano": 1350000, "Napoli": 975000}). Usa il metodo `.items()` (che restituisce una vista di coppie (chiave, valore) del dizionario) per stampare ogni città con la sua popolazione in formato "Città: Popolazione abitanti".
